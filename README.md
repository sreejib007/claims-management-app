# Claims Management App

A modern Angular-based claims management system with dashboard, policies, handlers, and AI chatbot features.

## Prerequisites

- **Node.js** (v18 or higher)
- **npm** (v9 or higher)
- **Angular CLI** (v19)

## Installation

```bash
# Clone the repository
git clone <repository-url>
cd claims-management-app

# Install dependencies
npm install
```

## Run the App

```bash
# Start development server
npm start
```

Open [http://localhost:4200](http://localhost:4200) in your browser.

## Build for Production

```bash
npm run build
```

Output will be in `dist/claims-management-app/`.

## Run Tests

```bash
npm test
```
